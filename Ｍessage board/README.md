# Message Board
透過 PHP 實作留言板網站。


[Demo page](http://mentor-program.co/mtr04group1/Kayla/week11/index.php)

**使用技術：**  
前端 -  
* HTML  
* CSS  
* RWD   
* JavaScript  

後端 -  
* PHP

**功能說明：**  
* 註冊會員功能
* 會員登入 / 登出機制
* 修改使用者暱稱
* 新增 / 刪除 / 編輯留言功能 
* 資訊安全防護：XSS、SQL Injection、處理明文密碼 
* 分頁：每頁顯示五筆留言
