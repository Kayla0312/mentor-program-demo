# Twitch Top Games 
串接 Twitch API，導覽列顯示 Twitch 排名前 5 名熱門遊戲，透過切換分頁，各別顯示出每個熱門遊戲的 20 個實況直播。  

[Demo page](https://kayla0312.github.io/mentor-program-demo/Twitch%20Top%20Games/)

**使用技術：**  
* HTML  
* CSS  
* RWD   
* JavaScript  

**功能說明：** 
* 顯示目前 5 筆熱門遊戲。
* 顯示每一熱門遊戲中，20 個實況頻道。
* 點選頻道，連結至實況遊戲頁面。
