# TO-DO LIST  
以＋ - ✓ x 待辦事項符號作為版面設計發想，設計可新增刪除與標記完成的 Todo List 版面。  

[Demo page](https://kayla0312.github.io/mentor-program-demo/To-Do%20List/)

**使用技術：**  
* HTML  
* CSS  
* RWD   
* JavaScript  

**功能說明：** 
* 新增/刪除事項  
* 標記已完成事項
